# 👋 Bienvenido

¿No sabes por dónde empezar? Aquí está tu guía:

## 🎯 ¿Qué Quieres Hacer?

### 1️⃣ **Usar la librería** (Instalación + primeros scripts)
→ Ve a `docs/QUICKSTART.md`  
⏱️ Tiempo: 5 minutos

### 2️⃣ **Entender todo en detalle**
→ Lee `README.md` en raíz (o `docs/README.md` para versión completa)  
⏱️ Tiempo: 15 minutos

### 3️⃣ **Ver ejemplos de código**
→ Ejecuta `python examples/basic_usage.py`  
⏱️ Tiempo: 5 minutos

### 4️⃣ **Subir a GitHub**
→ Sigue `docs/GITHUB_SETUP.md`  
⏱️ Tiempo: 20 minutos

### 5️⃣ **Usar en Windows**
→ Ve a `docs/WINDOWS_SETUP.md`  
⏱️ Tiempo: 10 minutos

### 6️⃣ **Contribuir o hacer cambios**
→ Lee `docs/DEVELOPMENT.md` + `docs/CONTRIBUTING.md`  
⏱️ Tiempo: 30 minutos

### 7️⃣ **Navegar toda la documentación**
→ Consulta `docs/INDEX.md`  
⏱️ Tiempo: 5 minutos

## 📚 Estructura del Proyecto

```
video_downloader_library/     ← Código fuente
docs/                         ← Documentación completa
examples/                     ← Ejemplos ejecutables
tests/                        ← Tests (preparado)
setup.py, requirements.txt    ← Configuración
```

Ver detalles en: `STRUCTURE.md`

## ⚡ Comandos Rápidos

```bash
# Instalar dependencias
pip install -r requirements.txt

# Instalar en modo desarrollo
pip install -e .

# Ver documentación
cat docs/QUICKSTART.md

# Ejecutar ejemplos
python examples/basic_usage.py

# Setup automático (Linux/Mac)
chmod +x setup.sh && ./setup.sh
```
